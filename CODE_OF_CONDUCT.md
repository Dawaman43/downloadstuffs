# Code of Conduct

We want a welcoming community for everyone.

- Be respectful and constructive
- Assume good intent
- No harassment or hate speech

We follow the Contributor Covenant (v2.1) as our baseline standard:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

If you need to report conduct issues, please open a private report using the contact in SECURITY.md.
