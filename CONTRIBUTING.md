# Contributing

Thanks for helping improve DownloadStuffs.

## Quick start

- Install deps: `bun install`
- Run dev: `bun run dev`
- Run tests: `bun run test`
- Lint/format: `bun run lint` and `bun run format`

## Pull requests

- Keep PRs focused (one feature/fix per PR)
- Include screenshots/recordings for UI changes
- If you add a new behavior, add or update a test when practical

## Code style

- TypeScript + React
- Prefer small, readable components
- Keep URL state in routes (search params) where it improves UX (back/forward/share)

## Reporting bugs

Please open an issue with:

- Steps to reproduce
- Expected vs actual behavior
- Browser + OS
- Console/network logs if relevant
